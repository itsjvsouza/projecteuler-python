# Project Euler
Repository dedicated to solving [Project Euler](https://projecteuler.net/) math problems using Python.

Current progress: 004/988 exercises solved.

### [Exercise 000](exercises-000-988/pe000.py)
Among the first 945 thousand square numbers, what is the sum of all the odd squares?
### [Exercise 001](exercises-000-988/pe001.py)
If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23. Find the sum of all the multiples of 3 or 5 below 1000.
### [Exercise 002](exercises-000-988/pe002.py)
By considering the terms in the Fibonacci sequence whose values do not exceed four million, find the sum of the even-valued terms.
### [Exercise 003](exercises-000-988/pe003.py)
What is the largest prime factor of the number 600851475143?
### [Exercise 004](exercises-000-988/pe004.py)
Find the largest palindrome made from the product of two 3-digit numbers.
